# 当前范围与非目标

## 当前范围

当前版本实现以下内容：

- 多人联机房间
- 房主选择战斗
- 8 人小队编制与空位补 Bot
- Boss 战斗脚本驱动的时间线模拟
- 玩家位置、朝向、移动、受击、死亡
- 强制位移与防击退
- 简化血量模型
- 简化失败判定
- 重开

## 当前不实现

- 完整职业技能系统
- 属性成长与装备系统
- 高精度伤害公式
- 护盾、减伤、仇恨、治疗量细分
- 单人模式
- 复杂 Bot 智能决策
- 通用关卡编辑器

## 占位接口

以下内容只保留接口：

- 玩家主动技能系统扩展位
- 战斗日志导出
- 回放系统
- 观战系统
- 战斗编辑工具
- 持久化排行榜

## 设计原则

- 复杂度集中到 `core`
- UI、网络和规则严格分层
- 战斗脚本直接描述时间线和效果
