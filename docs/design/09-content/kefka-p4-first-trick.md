# 凯夫卡P4：一运

本文档描述《最终幻想 XIV》凯夫卡 P4 一运练习内容的模拟设计。

本战斗规则实现于 `packages/content`，前端与服务端不得实现第二套战斗规则。  
当前版本只实现机制判定，不实现 Bot 跑法。

## 战斗范围

- 战斗 id：`kefka_p4_first_trick`
- 战斗名称：`凯夫卡P4：一运`
- 首领：`凯夫卡`
- 小队：8 人，标准 `MT/ST/H1/H2/D1/D2/D3/D4`
- 场地半径：`20m`
- 首领目标圈：`6m`
- 开场场外 `25m` 的左上 45 度位置生成 `卡奥斯`，右上 45 度位置生成 `艾克斯迪斯`

## 玄乎乎魔法

玄乎乎魔法读条 `4700ms`，读条期间同时显示两个范围：

- 雷范围：随机选择 `+45°` 或 `-45°` 斜向，显示两条宽 `10m`、长 `40m` 的平行直线。一侧中线距场中 `5m`，另一侧中线距场中 `15m`。假雷使用同斜向的互补组，即两侧距离互换为 `15m` 与 `5m`。
- 冰范围：显示两片对角 `90°`、半径 `20m` 扇形。假冰使用另一组对角扇形。

凯夫卡身上同步显示通用 `ringIndicator`：

- 内侧白环对应冰
- 外侧紫环对应雷
- 蓝色实心球表示真
- 红色问号球表示假

读条结束时按真假结果结算冰与雷。命中玩家受到普通机制伤害并获得 `3000ms` 易伤；已有易伤时受到任何机制伤害会死亡。

## 大十字

第一次大十字在 `9083ms` 赋予：

- `60000ms` 诅咒之嚎 + `51000ms` 加速度炸弹
- `76000ms` 加速度炸弹
- `51000ms` 或 `76000ms` 叉形闪电
- `51000ms` 或 `76000ms` 水属性压缩

第二次大十字在 `23997ms` 赋予：

- `69000ms` 诅咒之嚎 + `61000ms` 加速度炸弹
- `36000ms` 加速度炸弹
- `61000ms` 或 `36000ms` 叉形闪电
- `61000ms` 或 `36000ms` 水属性压缩

每次大十字读条有真假指示。假读条赋予的 Buff 以“假”规则结算，并在状态名中显示 `（假）`。  
`TH` 组与 `DPS` 组内各自随机分配四种 Buff 组。前两次大十字按 A/B 大类交替分配：`诅咒之嚎 + 加速度炸弹` 与 `加速度炸弹` 视为 A 组，`叉形闪电` 与 `水属性压缩` 视为 B 组；第一次获得 A 组的玩家第二次必定获得 B 组，第一次获得 B 组的玩家第二次必定获得 A 组。因此同一玩家不会在前两次大十字中同时获得叉形闪电和水属性压缩。  
全团随机一次雷水时长组。若第一次两个雷水 Buff 均为 `51000ms`，第二次两个雷水 Buff 均为 `61000ms`；若第一次两个雷水 Buff 均为 `76000ms`，第二次两个雷水 Buff 均为 `36000ms`。

大十字 Buff 结算：

- 诅咒之嚎：真 Buff 击杀面对持有者的其他玩家；假 Buff 击杀背对持有者的其他玩家。
- 叉形闪电：真 Buff 为自身中心 `8m` 圆形伤害；假 Buff 为自身中心 `8m` 三人分摊，少人时范围内玩家死亡。
- 水属性压缩：真 Buff 为自身中心 `8m` 三人分摊，少人时范围内玩家死亡；假 Buff 为自身中心 `8m` 圆形伤害。
- 加速度炸弹：在 Buff 到期前 `500ms` 采样持有者是否正在移动。真 Buff 采样时正在移动会在到期时死亡；假 Buff 采样时未移动会在到期时死亡。

## 混沌之水与混沌之炎

`5475ms` 卡奥斯随机读条海啸或烈焰；`20383ms` 必定读条另一种技能。

- 先海啸：`14175ms` 赋予 `84000ms` 混沌之水；`29083ms` 赋予 `45000ms` 混沌之炎。
- 先烈焰：`14175ms` 赋予 `60000ms` 混沌之炎；`29083ms` 赋予 `69000ms` 混沌之水。

混沌之水到期时锁定持有者当前位置，`5000ms` 后在锁定位置判定，判定前 `500ms` 显示预兆。  
真混沌之水为锁定位置内 `6m` 外 `40m` 环形伤害；假混沌之水为锁定位置 `6m` 圆形伤害。

混沌之炎到期时锁定持有者当前位置，`5000ms` 后在锁定位置判定，判定前 `500ms` 显示预兆。  
真混沌之炎为锁定位置 `6m` 圆形伤害；假混沌之炎为锁定位置内 `6m` 外 `40m` 环形伤害。

## 无之泛滥

最终大十字在 `38958ms` 赋予：

- 每名玩家随机获得 `15000ms` 生者之伤或死者之伤
- `TH` 组与 `DPS` 组内各自随机 `2` 人获得 `16000ms` 亚拉戈领域，另 `2` 人获得 `15000ms` 超越死亡

生者之伤与死者之伤吃到错误的暗黑光时移除并触发致死判定。假伤会对调错误暗黑光类型。

`46374ms` 艾克斯迪斯读条无之泛滥，读条期间在左右两侧随机生成蓝色三角形与紫色五边形。暗黑光在 `51565ms` 判定：

- 真读条：紫色五边形所在半场为生者暗黑光，蓝色三角形所在半场为死者暗黑光
- 假读条：两者对调

亚拉戈领域与超越死亡：

- 真亚拉戈领域：吃到错误暗黑光后，Buff 结束时全场死亡。
- 假亚拉戈领域：吃到致死伤害时抵挡并移除；自然结束时持有者死亡。
- 真超越死亡：吃到致死伤害时抵挡并移除；自然结束时持有者死亡。
- 假超越死亡：吃到错误暗黑光后，Buff 结束时全场死亡。
