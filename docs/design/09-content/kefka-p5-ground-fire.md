# 凯夫卡P5：地火

本文档描述《最终幻想 XIV》凯夫卡 P5 地火练习内容的模拟设计。

本战斗规则实现于 `packages/content`，前端与服务端不得实现第二套战斗规则。  
当前版本只实现机制判定，不实现 Bot 跑法。

## 战斗范围

- 战斗 id：`kefka_p5_ground_fire`
- 战斗名称：`凯夫卡P5：地火`
- 首领：`凯夫卡`
- 小队：8 人，标准 `MT/ST/H1/H2/D1/D2/D3/D4`
- 场地半径、首领目标圈和场地标点：沿用 [凯夫卡P5：癫狂交响曲](./kefka-p5-mad-symphony.md)
- 初始站位：8 人在南侧一字排开，面向北

## 地火起点

场地上方存在左右两条 `45°` 斜线，分别位于左上和右上。  
项目内场中坐标为 `(0, 0)`；若外部资料以 `(100, 100)` 表示场中，则项目坐标等于外部坐标减去 `(100, 100)`。

每条起始线满足：

- 线段长度为 `25√2m`
- 线段中垂线经过场中
- 线段中点到场中的中垂线长度为 `17.5√2m`
- 线段上平均分布 `6` 个半径 `6m` 的圆形起点，包含两端点，相邻起点间距为 `5√2m`
- 起点落在 `5m` 网格上

编号规则：

- 左线从正北侧起点向正西侧起点编号为左 `1-6` 号地火
- 右线从正北侧起点向正东侧起点编号为右 `1-6` 号地火
- 左右两侧各自随机排列 `1/4`、`2/5`、`3/6` 三组刷新顺序
- 同一组内的两个地火一定同时刷新

左侧 `n` 号地火起点为 `(-5n, -35 + 5n)`。  
右侧 `n` 号地火起点为 `(5n, -35 + 5n)`。  
其中 `n = 1..6`。

## 地火步进

每个地火序列从自身起点开始：

- 起点预兆显示 `4000ms`
- 序列开始后 `4500ms` 第一次判定，判定初始预兆位置
- 之后每 `500ms` 判定一次
- 左侧每次判定沿 `(+5, +5)` 前进，右侧每次判定沿 `(-5, +5)` 前进，斜向距离为 `5√2m`
- 总共判定 `7` 次，前进 `6` 次
- 每次判定后 `300ms` 显示当前地火范围

地火半径为 `6m`。  
玩家吃到任意地火立即死亡，死亡来源为“地火”。
