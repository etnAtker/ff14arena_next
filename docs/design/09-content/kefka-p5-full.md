# 凯夫卡P5：整合

本文档描述凯夫卡 P5 整合练习内容。  
本战斗规则实现于 `packages/content`，前端与服务端不得实现第二套战斗规则。

## 战斗范围

- 战斗 id：`kefka_p5_full`
- 战斗名称：`凯夫卡P5：整合`
- 首领：`凯夫卡`
- 小队：8 人，标准 `MT/ST/H1/H2/D1/D2/D3/D4`
- 场地、首领目标圈、标点和初始站位沿用凯夫卡 P5 现有配置

## 时间轴来源

整合战以 `tmp/P5施法.csv` 和 `tmp/P5buff.csv` 为时间来源。  
带读条时间的日志行按判定时点处理，读条开始时间由判定时点减去读条时间得到。

`T+0` 定义为第一条连续究极读条开始，对应日志中 `15:05.799 连续究极 3.70 秒` 的 `3.70s` 前。
