# 凯夫卡P3：二运

本文档描述《最终幻想 XIV》凯夫卡 P3 二运练习内容的模拟设计。

本战斗规则实现于 `packages/content`，前端与服务端不得实现第二套战斗规则。  
当前版本只实现机制判定，不实现 Bot 跑法。

## 战斗范围

- 战斗 id：`kefka_p3_second_trick`
- 战斗名称：`凯夫卡P3：二运`
- 首领：`凯夫卡`
- 小队：8 人，标准 `MT/ST/H1/H2/D1/D2/D3/D4`
- 场地半径：`20m`
- 首领目标圈：隐藏
- 场中没有固定 Boss，`卡奥斯` 与 `艾克斯迪斯` 作为表现用敌人标记分别追踪 `ST` 与 `MT`

## 时间轴

时间以地震读条结束后的 `0ms` 计。地震在战斗开始后 `1000ms` 开始读条，读条 `2000ms`。

- `0ms`：随机分配 `3` 名第一目标、`3` 名第二目标、`2` 名第三目标，并赋予同持续时间的混沌之土
- `9800ms`：场外随机 45 度倍数位置生成凯夫卡，随后执行一次响亮亮耳光
- `16500ms`：开始读条黑洞，读条 `2700ms`，结束后生成第一组黑洞
- `31900ms`：艾克斯迪斯读条暴雷，读条 `4700ms`，结束后判定两次最近目标圆形范围
- `39900ms`：卡奥斯读条诅咒敕令，读条 `4700ms`，开始时随机锁定一名玩家方向
- `40200ms`：再执行一次响亮亮耳光
- `49300ms`：生成第二组黑洞，并执行三轮持续连线射线
- `63500ms`：卡奥斯再次读条诅咒敕令；同时场外生成凯夫卡准备本色出演的我
- `69500ms`：艾克斯迪斯再次读条暴雷
- `81000ms`：生成第三组黑洞，重复第二组黑洞的三轮持续连线射线
- `100000ms`：场外再次生成凯夫卡，`106000ms` 开始读条响亮亮耳光
- `104000ms`：卡奥斯随机读条经度聚爆或纬度聚爆，参考 P3 一运的聚爆顺序结算两轮扇形

所有 AOE 在结算前 `500ms` 显示预兆。黑洞连线源使用固定坐标源连线，允许玩家穿线接走。

## 状态

- 第一目标：状态 id `kefka_p3_second_first_target`，图标来源 `Status/3004`
- 第二目标：状态 id `kefka_p3_second_second_target`，图标来源 `Status/3005`
- 第三目标：状态 id `kefka_p3_second_third_target`，图标来源 `Status/3006`
- 混沌之土：状态 id `kefka_p3_second_chaos_earth`，逻辑来源 `Status/5454`，中文文本由项目手写
- 无之侵蚀 1 层：状态 id `kefka_p3_second_void_erosion_1`，逻辑来源 `Status/5452`
- 无之侵蚀 2 层：状态 id `kefka_p3_second_void_erosion_2`，使用 1 层图标 id 加 `1`
- 无之腐蚀：状态 id `kefka_p3_second_void_corrosion`，逻辑来源 `Status/5453`

混沌之土在持有者受到即将导致死亡的机制伤害时消耗，使持有者不死，并在 `1000ms` 后对其他所有玩家造成一次机制伤害并附加 `3s` 易伤。

无之侵蚀首次获得为 1 层，第二次替换为 2 层，第三次替换为无之腐蚀。持有无之腐蚀后再次受到黑洞射线命中时死亡。

目标状态赋予时同步生成头顶标点：

- 第一目标 3 人显示黄色 `1/2/3`
- 第二目标 3 人显示紫色 `1/2/3`
- 第三目标 2 人显示红色 `1/2`

## 机制口径

`卡奥斯` 与 `艾克斯迪斯` 每秒按目标圈偏移追踪 `ST` 与 `MT`，表现用敌人标记显示当前面向。正常跟随时卡奥斯朝向 `ST`，艾克斯迪斯朝向 `MT`；锁定目标类效果会将对应小 Boss 面向改为锁定方向。读条开始后对应小 Boss 停止移动，直到该机制完成后 `1000ms`。

响亮亮耳光在凯夫卡伸手一侧的相反半场依次生成三个半径 `13m` 圆形范围。圆形范围坐标以场中为原点，并以场中指向场外凯夫卡的方向作为正北；右手使用本地 `x=-10`，左手使用本地 `x=10`，本地 `y` 依次为 `-10/0/10`。右手后续为 `8` 人分摊扇形，左手后续为随机 `1T/1H/1DPS` 三个扇形。

黑洞生成在场中 `17m` 正点。第一组先由一个黑洞连一名玩家并射线，之后剩余两个黑洞连线并射线。第二组和第三组三个黑洞同时连线，连续射线三轮后消失。黑洞连线允许同一玩家同时持有多根线。

暴雷读条结束后以艾克斯迪斯最近玩家为中心结算 `5m` 圆形范围，`3000ms` 后再次选择最近玩家结算。`MT/ST` 命中为普通机制伤害，非 T 命中为即死类机制伤害。

诅咒敕令在读条开始时随机选择一名机制有效玩家，并锁定卡奥斯指向该玩家的方向，读条结束后对该方向释放 `180°`、半径 `30m` 的即死扇形。

本色出演的我从场外凯夫卡向场中心方向释放宽 `10m`、长 `50m` 的矩形即死范围。

经度聚爆/纬度聚爆在读条开始时锁定卡奥斯当前位置，并锁定该位置指向当时 `ST` 的方向作为卡奥斯面向。读条 `5000ms`，第一轮在读条结束时结算，第二轮在 `2000ms` 后结算；每轮为两片相对 `90°`、半径 `60m` 的即死扇形。经度聚爆先前后再左右，纬度聚爆先左右再前后。
