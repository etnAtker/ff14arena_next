# 凯夫卡P3：二运

本文档描述《最终幻想 XIV》凯夫卡 P3 二运练习内容的模拟设计。

本战斗规则实现于 `packages/content`，前端与服务端不得实现第二套战斗规则。  
当前版本只实现机制判定，不实现 Bot 跑法。

## 战斗范围

- 战斗 id：`kefka_p3_second_trick`
- 战斗名称：`凯夫卡P3：二运`
- 首领：`凯夫卡`
- 小队：8 人，标准 `MT/ST/H1/H2/D1/D2/D3/D4`
- 场地半径：`20m`
- 首领目标圈：隐藏
- 场中没有固定 Boss，`卡奥斯` 与 `艾克斯迪斯` 作为表现用敌人标记分别追踪当前配置指定的目标
- 机制房间选项：`MT拉艾克斯迪斯`，默认关闭；开启时艾克斯迪斯追踪 `MT`、卡奥斯追踪 `ST`，关闭时艾克斯迪斯追踪 `ST`、卡奥斯追踪 `MT`

## 时间轴

时间以开战瞬间后的绝对战斗时间计。地震在 `1000ms` 开始读条，读条 `2000ms`，因此地震结束点名为 `3000ms`。

- `3000ms`：随机分配 `3` 名第一目标、`3` 名第二目标、`2` 名第三目标，并赋予同持续时间的混沌之土
- `12800ms`：场外随机 45 度倍数位置生成凯夫卡，随后执行一次响亮亮耳光
- `19400ms`：开始读条黑洞，读条 `2700ms`，结束后生成第一组黑洞
- `34800ms`：艾克斯迪斯读条暴雷，读条 `4700ms`，结束后判定两次最近目标圆形范围
- `42800ms`：卡奥斯读条诅咒敕令，读条 `4700ms`，开始时随机锁定一名玩家方向；同时场外再次生成凯夫卡，`44300ms` 开始读条响亮亮耳光
- `52600ms`：生成第二组黑洞，并执行三轮持续连线射线
- `70000ms`：卡奥斯再次读条诅咒敕令；同时场外生成凯夫卡准备本色出演的我
- `76000ms`：艾克斯迪斯再次读条暴雷
- `86900ms`：生成第三组黑洞，重复第二组黑洞的三轮持续连线射线
- `106100ms`：场外再次生成凯夫卡，`112100ms` 开始读条响亮亮耳光
- `110600ms`：卡奥斯随机读条经度聚爆或纬度聚爆，参考 P3 一运的聚爆顺序结算两轮扇形
- `120400ms`：生成第四组黑洞，先由两个相邻黑洞连线并在 `127500ms` 射线，再由剩余一个黑洞连线并在 `134600ms` 射线
- `127500ms`：两个相邻黑洞射线判定时，场外再次生成凯夫卡，`129900ms` 开始读条 `4700ms` 的本色出演的我
- `134600ms`：最后一次本色出演的我与剩余单黑洞射线同 tick 判定
- `140100ms`：最后一次场外凯夫卡标记消失
- `141100ms`：同一个凯夫卡在场中生成仅用于绘图的方向标记，面向最后一次场外凯夫卡消失位置的反方向
- `145100ms`、`148100ms`：记录所有机制有效玩家当前位置并生成半径 `5m` 脚下圆形预警，`3000ms` 后按记录中心判定即死
- `145100ms`、`151100ms`：分别点名一个分摊目标，一次点 `T/H` 组，一次点 `DPS` 组，顺序随机；分摊标识为跟随目标玩家的黄色 `4m` 圆圈，圈的上、右、下、左四个方向显示指向中心的箭头；`6000ms` 后按目标当前位置半径 `4m` 判定，少于 `4` 人时范围内玩家即死，成功参与者获得 `2500ms` 易伤，分摊标识在判定后消失
- `151100ms`、`152100ms`、`154100ms`、`155100ms`：按箭头正前定义左右，在场中右、左、右、左 `8m` 位置生成半径 `4m` 塔并立即判定；塔内少于 `2` 人时全员即死，否则塔内玩家获得 `2500ms` 易伤；塔显示至少保留 `500ms`
- `160100ms`：检查所有机制有效玩家是否处于移动状态，不移动者即死
- `160600ms`：在两次分摊实际判定位置显示半径 `4m` 石笋范围预兆
- `161100ms`：在两次分摊实际判定位置重新结算半径 `4m` 即死范围
- `165000ms`：战斗完成

除终盘脚下圈使用 `3000ms` 圆形预警外，其他 AOE 在结算前 `500ms` 显示预兆。黑洞连线源使用固定坐标源连线，允许玩家穿线接走。

## 状态

- 第一目标：状态 id `kefka_p3_second_first_target`，图标来源 `Status/3004`
- 第二目标：状态 id `kefka_p3_second_second_target`，图标来源 `Status/3005`
- 第三目标：状态 id `kefka_p3_second_third_target`，图标来源 `Status/3006`
- 混沌之土：状态 id `kefka_p3_second_chaos_earth`，逻辑来源 `Status/5454`，中文文本由项目手写
- 无之侵蚀：状态 id `kefka_p3_second_void_erosion_1`，逻辑来源 `Status/5452`
- 无之腐蚀：状态 id `kefka_p3_second_void_corrosion`，逻辑来源 `Status/5453`

混沌之土在持有者受到即将导致死亡的机制伤害时消耗，使持有者不死，并在 `1000ms` 后对其他所有玩家造成一次机制伤害并附加 `3s` 易伤。

无之侵蚀首次受到黑洞射线命中时获得，第二次命中直接替换为无之腐蚀。持有无之腐蚀后再次受到黑洞射线命中时死亡。

目标状态赋予时同步生成头顶标点：

- 第一目标 3 人显示黄色 `1/2/3`
- 第二目标 3 人显示紫色 `1/2/3`
- 第三目标 2 人显示红色 `1/2`

## 机制口径

`卡奥斯` 与 `艾克斯迪斯` 每秒按目标圈偏移追踪当前配置指定的目标，表现用舞台角色显示当前面向。默认配置下卡奥斯朝向 `MT`，艾克斯迪斯朝向 `ST`；开启 `MT拉艾克斯迪斯` 后两者目标互换。锁定目标类效果会将对应小 Boss 面向改为锁定方向。读条开始后对应小 Boss 停止移动，直到该机制完成后 `1000ms`。

响亮亮耳光在凯夫卡伸手一侧的相反半场依次生成三个半径 `13m` 圆形范围。圆形范围坐标以场中为原点，并以场中指向场外凯夫卡的方向作为正北；右手使用本地 `x=-10`，左手使用本地 `x=10`，本地 `y` 依次为 `-10/0/10`。右手后续为 `8` 人分摊扇形，左手后续为随机 `1T/1H/1DPS` 三个扇形。

第一次响亮亮耳光后的场外凯夫卡在第一次暴雷首击后稍晚消失，并在第一次诅咒敕令读条开始时重新出现。后续没有明确消失日志的场外凯夫卡按下一次出现前 `2000ms` 消失处理。最后一次场外凯夫卡在 `140100ms` 消失，并在 `141100ms` 作为同一个凯夫卡于场中重新出现。

黑洞生成在场中 `17m` 正点。第一组先由一个黑洞连一名玩家并在生成后 `7100ms` 射线，之后剩余两个黑洞连线并再经过 `7100ms` 射线。第二组和第三组三个黑洞同时连线，生成后 `7100ms` 首射，之后每 `5100ms` 连续射线两轮，第三轮后消失。第四组在 `120400ms` 生成，先由两个相邻黑洞连线并在 `127500ms` 射线消失，同时触发最后一次场外凯夫卡出现；然后剩余一个黑洞立即连线并在 `134600ms` 射线消失。黑洞连线允许同一玩家同时持有多根线。

暴雷读条结束后以艾克斯迪斯最近玩家为中心结算 `5m` 圆形范围，`3000ms` 后再次选择最近玩家结算。`MT/ST` 命中为普通机制伤害，非 T 命中为即死类机制伤害。

诅咒敕令在读条开始时随机选择一名机制有效玩家，并锁定卡奥斯指向该玩家的方向，读条结束后对该方向释放 `180°`、半径 `30m` 的即死扇形。

本色出演的我从场外凯夫卡向场中心方向释放宽 `10m`、长 `50m` 的矩形即死范围。后半段追加的本色出演的我在 `127500ms` 凯夫卡出现，`129900ms` 开始读条，读条 `4700ms`，并在 `134600ms` 与最后一次黑洞射线同 tick 判定。

终盘凯夫卡方向标记在 `141100ms` 于场中生成，只作为绘图标记，不参与机制判定。箭头方向固定为最后一次场外凯夫卡消失位置的反方向。之后两轮脚下圈在生成时记录玩家当前位置，预警 `3000ms` 后按记录中心判定；两次分摊一次点 `T/H` 组、一次点 `DPS` 组，顺序随机，分摊标识为跟随目标玩家的黄色 `4m` 圆圈和四个向内箭头；成功分摊和终盘塔都会赋予 `2500ms` 易伤，易伤重叠时即死。终盘塔按箭头正前定义左右，右、左、右、左依次生成，判定立即生效，塔显示保留 `500ms`。`160100ms` 的移动检查只检查 `moveState.moving`，`161100ms` 的返还范围使用两次分摊实际判定位置，并只在判定前 `500ms` 显示石笋范围预兆。

经度聚爆/纬度聚爆在读条开始时锁定卡奥斯当前位置，并锁定该位置指向当时卡奥斯追踪目标的方向作为卡奥斯面向。读条 `4700ms`，第一轮在读条结束时结算，第二轮在 `2000ms` 后结算；每轮为两片相对 `90°`、半径 `60m` 的即死扇形。经度聚爆先前后再左右，纬度聚爆先左右再前后。
