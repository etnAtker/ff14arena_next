# 凯夫卡P5：洪水

本文档描述《最终幻想 XIV》凯夫卡 P5 洪水练习内容的模拟设计。

本战斗规则实现于 `packages/content`，前端与服务端不得实现第二套战斗规则。  
当前版本只实现机制判定，不实现 Bot 跑法。

## 战斗范围

- 战斗 id：`kefka_p5_flood`
- 战斗名称：`凯夫卡P5：洪水`
- 首领：`凯夫卡`
- 小队：8 人，标准 `MT/ST/H1/H2/D1/D2/D3/D4`
- 场地半径、首领目标圈和场地标点：沿用 [凯夫卡P5：地火](./kefka-p5-ground-fire.md)
- 初始站位：沿用凯夫卡 P5 地火，8 人在南侧一字排开，面向北

## 时间轴

时间以开战瞬间起算。

| 时间      | 事件       | 数据                                    |
| --------- | ---------- | --------------------------------------- |
| `1000ms`  | 洪水读条   | 凯夫卡读条 `5000ms`，同时显示第一轮预兆 |
| `2000ms`  | 第二轮预兆 | 显示 `1300ms`，与第一轮预兆重叠         |
| `3000ms`  | 第三轮预兆 | 显示 `1300ms`，与第二轮预兆重叠         |
| `4000ms`  | 第四轮预兆 | 显示 `1300ms`，与第三轮预兆重叠         |
| `7000ms`  | 第一轮判定 | 判定后显示本轮范围 `300ms`              |
| `8000ms`  | 第二轮判定 | 判定后显示本轮范围 `300ms`              |
| `9000ms`  | 第三轮判定 | 判定后显示本轮范围 `300ms`              |
| `10000ms` | 第四轮判定 | 判定后显示本轮范围 `300ms`              |
| `11000ms` | 战斗完成   | 结束本练习内容                          |

## 洪水范围

洪水范围参照凯夫卡 P4 真假雷的雷范围：

- 每轮为两条宽 `10m`、长 `40m` 的平行矩形范围
- 矩形方向随机为 `+45°` 或 `-45°`
- 同一方向有两组互补平行线组合，一侧距场中 `5m`，另一侧距场中 `15m`
- 四轮预兆不重复使用四种范围组合
- 相同方向不会连续刷新

玩家吃到任意洪水立即死亡，死亡来源为“洪水”。
