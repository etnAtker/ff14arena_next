# 首个战斗定义

## 战斗标识

- `id`: `opening_two_rounds`
- `name`: `双轮组合练习`

## 基础规则

- 场地为半径 `20m` 的圆形区域
- Boss 目标环半径为 `5m`
- 每个玩家与 Bot 最大血量为 `10000`
- 越过场地边界后立即死亡

## Boss 技能

### 钢铁

- 以 Boss 为中心的圆形 AOE
- 半径 `5m`
- 伤害 `5000`

### 月环

- 以 Boss 为中心的环形 AOE
- 内径 `6m`
- 外径 `15m`
- 伤害 `5000`

### 分摊

- 点名 `H1` 与 `H2`
- 分摊圈半径 `5m`
- 总伤害 `10000`
- 单人承伤时受到 `10000`
- 多人承伤时均摊总伤害
- 命中后附加 `受伤加重`

### 分散

- 点名 8 名成员
- 分散圈半径 `1m`
- 每次命中伤害 `2500`
- 命中后附加 `受伤加重`

### 受伤加重

- 持续时间 `1s`
- 持有期间受到的后续伤害乘以 `10`

## 释放顺序

- Boss 共释放两轮技能
- 每轮由一个范围技能和一个点名技能组成
- 范围技能集合为：`钢铁`、`月环`
- 点名技能集合为：`分摊`、`分散`
- 第一轮随机抽取一组组合
- 第二轮使用剩余未出现的组合

## 读条与判定

- Boss 在每轮开始时显示读条
- 每轮先结算范围技能，再结算点名技能
- 两次结算之间固定间隔 `100ms`

## Bot 跑法

### 钢铁

- 所有 Bot 目标站位半径为 `6m`

### 月环

- 所有 Bot 目标站位半径为 `5m`

### 分摊

- 左侧分摊组：`MT` `H1` `D1` `D3`
- 右侧分摊组：`ST` `H2` `D2` `D4`

### 分散

- 上：`MT`
- 右上：`D4`
- 右：`H2`
- 右下：`D2`
- 下：`ST`
- 左下：`D1`
- 左：`H1`
- 左上：`D3`

## HUD 显示接口

战斗写入以下显示接口：

- `bossCastBar`
- `battleMessage`
- `recentFailureReason`

`recentFailureReason` 显示聚合后的失败原因文本。  
具体显示文案由战斗脚本定义。

## 失败条件

失败结果标记条件如下：

- 任意成员越界死亡
- 任意成员因吃错机制死亡

## 成功条件

两轮技能全部结算完成后，战斗结束并给出最终结果：

- 未写入失败结果标记时，最终结果为成功
- 已写入失败结果标记时，最终结果为失败，并显示聚合后的失败原因列表
