本目录不描述当前实现细节，当前实现请查看 `docs/implementation`。

## 2. 移动同步后续项

当前移动链路已经改为：

- 客户端本地模拟普通移动
- 客户端通过 `sim:input-frame` 上报位姿样本
- 服务端在 `waiting` 与 `running` 中共用同一套位姿样本处理链
- 远端角色在渲染层做平滑追逐

当前可选方向包括：

- 更细致的远端朝向平滑
- 更完整的强制位移、瞬移和特殊地形位移语义
- 基于轨迹历史的机制回溯判定
- 更细的网络抖动自适应策略
