## 2. 高延迟移动优化

当前已落地第一版高延迟移动优化：

- 客户端定时做轻量时间同步
- 连续移动输入携带 `issuedAtServerTimeEstimate`
- 服务端按最近移动历史与估计生效时刻做受控补偿
- 本机角色做即时乐观移动
- 远端角色在渲染层做平滑追逐
- 击退、越界修正等强修正通过硬修正事件处理

当前可选方向包括：

- 更完整的本地未确认输入重放与权威重演
- 更细致的远端朝向平滑
- 更完整的强制位移、瞬移和特殊地形位移语义
- 更细的网络抖动自适应策略
