## 2. 高延迟移动优化

当前客户端尚未实现完整的高延迟移动优化方案。  
该主题单独列出，不与当前实现文档混写。

当前可选方向包括：

- 本地位置预测与权威修正
- 远端角色平滑插值
- 影子追逐式位置同步

“影子追逐式”位置同步的候选方案如下：

- 本地角色即时移动
- 客户端上报目标位置或连续位置帧
- 服务端将目标位置视为影子点
- 服务端权威角色按合法移速追逐影子点
- 远端客户端平滑追逐权威或影子位置
- 瞬移、击退、强制位移等特殊位移通过非平滑修正处理
